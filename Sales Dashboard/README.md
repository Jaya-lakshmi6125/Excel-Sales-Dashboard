\# 📊 Excel Sales Dashboard



\## 📌 Project Overview



This project presents an interactive Sales Dashboard built using Microsoft Excel. It helps analyze sales performance, revenue trends, customer insights, and product performance through charts, pivot tables, and slicers.



\## 🛠️ Tools Used



\- Microsoft Excel

\- Pivot Tables

\- Pivot Charts

\- Slicers

\- Conditional Formatting



\## 📂 Dataset



Sales Dataset



\## 📊 Dashboard Features



\- Total Sales

\- Total Profit

\- Total Orders

\- Monthly Sales Trend

\- Region-wise Sales

\- Category-wise Performance

\- Top Products

\- Interactive Slicers
